## 介绍

这是一个vue UI 组件库

## 开始使用

1. 设置

使用本框架前，请在CSS中开启border-box

```css
* { border-box: box-sizing; }
*::before { border-box: box-sizing; }
*::after { border-box: box-sizing; }
```
2. 安装
```shell
npm install guru-ui --save
```
